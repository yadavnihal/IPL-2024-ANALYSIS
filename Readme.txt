# 🏏 IPL 2024 Analysis Project

## 📊 Project Overview
This project presents a complete data analysis of the **Indian Premier League (IPL) 2024 season**, utilizing data science techniques to turn raw data into actionable insights. It covers the journey from data collection and cleaning to building advanced visualizations using **Python, Excel, and Power BI**.

---

## 🚀 Steps Involved

### 1️⃣ Collecting & Cleaning Data
- **Data Source:** IPL dataset from [GitHub Repository](https://github.com/ritesh-ojha/IPL-DATASET/tree/main).
- **Filtering:** Applied Excel filters to extract only the IPL 2024 season records.
- **Python Cleaning:** Used Python scripts (with ChatGPT assistance) to automate data cleaning and ensure consistent, reliable datasets.

### 2️⃣ Importing & Transforming Data in Power BI
- **Import:** Loaded the cleaned CSV files into **Power BI Desktop**.
- **Transformations Performed:**
  - Renamed columns for better readability.
  - Dropped unnecessary columns and duplicate records.
  - Ensured data types were properly set for analysis.

### 3️⃣ Data Modeling in Power BI
- Built relationships between multiple tables (matches, players, teams, performance stats) to establish a comprehensive data model, enabling cross-table analysis.

### 4️⃣ Creating Measures
- Developed critical performance measures such as:
  - 🏏 **Batting Average**
  - 🚀 **Strike Rate**
  - 🎯 **Bowling Economy**
  - Plus role-specific metrics to enrich the depth of the analysis.

### 5️⃣ Data Visualization
- Created interactive dashboards showcasing player performance across different roles:
  - **Openers**
  - **Middle-order Batsmen**
  - **Lower-order Anchors**
  - **All-rounders**
  - **Bowlers**

### 6️⃣ Enhancing the Analysis
- Polished the dashboards with slicers, drill-downs, and refined visuals to enable deeper insights and easier exploration.

---

## ✅ Conclusion
This project highlights how to leverage **data science and BI tools** to analyze sports data effectively. By combining **Excel**, **Python**, and **Power BI**, it delivers a complete pipeline that transforms raw IPL data into compelling, story-driven dashboards—offering a clear view into team dynamics and player performance trends throughout IPL 2024.
